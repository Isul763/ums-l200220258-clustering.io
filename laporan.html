Pendahuluan

Analisis ini dilakukan untuk mengelompokkan pesan teks berdasarkan isi menggunakan teknik clustering. Data dianalisis menggunakan TF-IDF dan algoritma K-Means untuk menemukan pola utama dalam pesan.

Proses Analisis

  1. Preprocessing: Data teks diubah menjadi representasi numerik menggunakan TF-IDF dengan 37 fitur unik.
  2. Clustering: Pesan dikelompokkan menjadi 3 cluster menggunakan algoritma K-Means.

Hasil Clustering

Cluster Kata Kunci
Cluster 1 offline, dari, tidak
Cluster 2 di, kelas, zoom
Cluster 3 pak, terimakasih, baik

Kesimpulan

Clustering berhasil mengelompokkan pesan menjadi 3 cluster, masing-masing dengan kata kunci yang mencerminkan tema utama. Analisis ini menunjukkan pola dominan yang dapat digunakan untuk memahami isi pesan secara lebih baik.
